# Game of Life

## Description

This is an implementation of Conway's Game of Life Cellular Automaton in cpp using the raylib library.

## Controls

-- Press 'R' to randomize the board (resets the board to the initial state)
-- Press 'Space' to start/stop the simulation
-- Press 'C' to clear the board
